# first-game

# Concept
**Tower Defense game**
# Gameplay
Монстры идут по пути уровня, а игрок должен помешать им дойти до конца. Для этого игрок может покупать и ставить защитные башни, которые атакуют монстров. За убийство монстров игрок будет получать золото.
